# Setup Instructions

## Development

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:4321`

## Building

Build for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

For information on how to contribute entries, see [README.md](README.md).
